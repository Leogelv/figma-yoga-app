# Документация по страницам практик

## Общее описание

В приложении реализованы страницы для отображения практик и детальной информации о них. Практики выводятся на основе выбора пользователя в квиз-флоу и могут фильтроваться по различным параметрам.

## Структура страниц

### 1. Страница результатов квиза (`/quiz/results`)

Эта страница отображает отфильтрованный список практик, соответствующих выбору пользователя в квизе.

#### Основные компоненты:
- Заголовок с названием раздела
- Список карточек практик (компонент `PracticeCard`)
- Кнопка "Изменить параметры" для возврата к настройке фильтров

#### Функциональность:
- Загрузка и отображение списка отфильтрованных практик
- Возможность выбрать практику для просмотра деталей
- Возможность вернуться к настройке параметров квиза

### 2. Страница детальной информации о практике (`/practice/[id]`)

Эта страница отображает полную информацию о выбранной практике.

#### Основные компоненты:
- Изображение практики (большое, вверху страницы)
- Заголовок и описание практики
- Информация о длительности и сложности
- Список целей практики
- Информация об инструкторе
- Кнопка "Начать практику" для запуска выполнения практики

#### Функциональность:
- Отображение полной информации о практике
- Имитация запуска/паузы практики
- Возможность вернуться к списку практик

## Процесс разработки

### Изображения для практик

Изображения для практик хранятся в директории `/public/images/practices/` и соответствуют полю `imageUrl` в объектах практик из файла `data/practices.ts`.

Для локальной разработки созданы заглушки изображений с помощью скрипта:
```bash
node scripts/placeholder-images.js
```

### Карточка практики (`PracticeCard`)

Компонент `PracticeCard` используется для отображения краткой информации о практике в списке:
- Изображение практики
- Название и краткое описание
- Длительность практики
- Уровень сложности
- Теги практики

### Функция фильтрации практик

В файле `data/practices.ts` реализована функция `filterPractices`, которая принимает параметры фильтрации и возвращает отфильтрованный список практик.

```typescript
// Пример использования функции фильтрации
const filteredPractices = filterPractices({
  practiceType: 'body',
  difficulty: 'beginner',
  goals: ['расслабление', 'гибкость']
});
```
